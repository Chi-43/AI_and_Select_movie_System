# AI对话功能使用说明

## 功能概述
已将电影推荐系统的"关于"页面替换为AI对话界面，使用DeepSeek API提供智能电影对话服务。

## 主要特性

### 1. 智能对话功能
- 🤖 **AI电影助手**：基于DeepSeek的智能对话系统
- 🎬 **电影推荐**：根据用户偏好推荐个性化电影
- 📝 **剧情分析**：分析电影剧情、角色和主题
- 🎭 **演员导演介绍**：提供电影相关人物信息
- 🌟 **观影建议**：提供观影时间和场合建议

### 2. 界面设计
- 🎨 **现代化UI**：采用渐变色彩和卡片设计
- 🌙 **深色模式**：支持深色/浅色主题切换
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🎯 **直观操作**：简洁明了的用户界面

### 3. 侧边栏功能
- 🔍 **电影搜索**：快速搜索电影数据库
- 🏆 **热门电影**：显示TOP5热门电影
- ⚙️ **对话设置**：调整AI模型和参数
- 📊 **电影统计**：显示电影总数和类型数量

## 安装和配置

### 1. 环境要求
- Python 3.8+
- Node.js 14+
- Django 4.0+
- Vue.js 3.0+

### 2. 后端配置
1. 安装依赖：
   ```bash
   cd backend
   pip install -r requirements.txt
   pip install python-dotenv requests
   ```

2. 配置DeepSeek API密钥：
   - 在`backend/.env`文件中设置：
     ```
     DEEPSEEK_API_KEY=your-deepseek-api-key-here
     ```
   - 或者设置环境变量：
     ```bash
     export DEEPSEEK_API_KEY=your-deepseek-api-key-here
     ```

3. 启动后端服务器：
   ```bash
   python manage.py runserver
   ```

### 3. 前端配置
1. 安装依赖：
   ```bash
   cd frontend
   npm install
   ```

2. 启动前端服务器：
   ```bash
   npm run serve
   ```

## 使用指南

### 1. 访问AI对话界面
1. 打开浏览器访问：`http://localhost:8080`
2. 点击导航栏的"🤖 AI对话"链接
3. 进入AI电影助手界面

### 2. 开始对话
1. **输入问题**：在底部输入框输入您的问题
   - 示例："推荐一部科幻电影"
   - 示例："《肖申克的救赎》讲的是什么？"
   - 示例："介绍克里斯托弗·诺兰的电影风格"

2. **快速提问**：使用预设的快速问题按钮
   - 热门推荐
   - 经典电影
   - 家庭观影
   - 悬疑推理

3. **使用示例**：点击示例按钮快速输入常见问题

### 3. 功能操作
1. **清空对话**：点击"🗑️ 清空对话"按钮
2. **切换主题**：点击"🌙 深色模式"/"☀️ 浅色模式"按钮
3. **折叠侧边栏**：点击侧边栏的"◀️"/"▶️"按钮
4. **复制回复**：点击AI回复下方的"📋 复制"按钮
5. **重新生成**：点击"🔄 重新生成"按钮
6. **点赞回复**：点击"🤍 点赞"按钮

### 4. 侧边栏功能
1. **电影搜索**：在搜索框输入电影名称、类型或国家
2. **查看热门电影**：点击热门电影列表查看详情
3. **调整AI设置**：
   - 选择AI模型：DeepSeek Chat / DeepSeek Coder
   - 调整温度参数：控制AI的创造性
   - 启用/禁用自动电影推荐

## API接口说明

### 1. AI对话接口
- **URL**: `POST /api/ai/chat/`
- **参数**:
  ```json
  {
    "message": "用户消息",
    "model": "deepseek-chat",
    "temperature": 0.7
  }
  ```
- **响应**:
  ```json
  {
    "response": "AI回复内容",
    "model": "使用的模型",
    "tokens_used": 消耗的token数量
  }
  ```

### 2. 电影推荐接口
- **URL**: `POST /api/ai/recommendations/`
- **参数**:
  ```json
  {
    "preferences": {
      "genre": "科幻",
      "year": "2020",
      "country": "美国"
    },
    "count": 5
  }
  ```

### 3. 电影分析接口
- **URL**: `POST /api/ai/analysis/`
- **参数**:
  ```json
  {
    "movie_title": "盗梦空间",
    "analysis_type": "剧情"
  }
  ```

## 故障排除

### 1. API密钥问题
**症状**: AI回复显示"DeepSeek API密钥未配置"
**解决方案**:
1. 检查`.env`文件中的`DEEPSEEK_API_KEY`设置
2. 确保环境变量已正确加载
3. 重启后端服务器

### 2. 网络连接问题
**症状**: AI回复显示"网络连接错误"
**解决方案**:
1. 检查网络连接
2. 确保后端服务器正常运行
3. 检查CORS配置

### 3. 前端编译错误
**症状**: 前端页面无法加载
**解决方案**:
1. 检查Node.js版本
2. 重新安装依赖：`npm install`
3. 清除缓存：`npm cache clean --force`

## 注意事项

1. **API限制**: DeepSeek API有调用频率限制，请合理使用
2. **数据安全**: 不要在前端暴露API密钥
3. **性能优化**: 大量对话时建议启用本地缓存
4. **用户体验**: 首次加载可能需要较长时间
